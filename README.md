# CODSOFT
C++ internship projects (Number Guessing Game, Tic-Tac-Toe Game, Library Management System)
This repository contains the completion of C++ tasks assigned from CodSoft.
It includes the following tasks;
Guessing Number Game:The program generates a random number and prompts the user to guess it. The project demonstrates the use of loops, conditional statements, random number generation, and user interaction.
Tic-Tac-Toe Game: It allows players to take turns marking positions on a 3×3 game board with X and O. The program checks for winning combinations, draw conditions, and displays the final result. The project showcases arrays, loops, conditional statements, and game logic implementation.
Library Management System: It enables users to view available books, issue and return books, search records, and calculate fines for overdue returns. The project demonstrates the use of arrays, menu-driven programming, conditional statements, and data management techniques.
